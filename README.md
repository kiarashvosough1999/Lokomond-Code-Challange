### Current Features:

- Pure SwiftUI
- Unit Tests
- Offline First
- Used **`MVVM`** + **`Clean`** Architecture Pattern
- Fully decoupled layers
- Modularized
- Used Dependency Injection
- CoreData Store
- Several Custom UI Components
